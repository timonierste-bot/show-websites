# Sonderprojekte

Dieser Ordner enthaelt Preview- oder Showcase-Projekte, die bewusst **nicht** zur Referenzliste Top Preview 100 gehoeren.

Beispiele:

- individuelle Kundenvorschauen
- separat entstandene Vertriebsprojekte
- interne Sonderfaelle ausserhalb der Marketing-Top-100

Wichtig:

- showcases/ ist fuer die Top-100-Referenz gedacht
- sonderprojekte/ ist fuer abweichende Einzelprojekte gedacht
