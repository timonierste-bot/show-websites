# Becker Bedachungen

Erste Show-Website fuer das Segment Handwerk.

## Ziel

Diese Demo soll zeigen, wie eine verkaufsstarke Handwerker-Website fuer einen Dachdeckerbetrieb aufgebaut sein kann:

- lokal und vertrauenswuerdig
- modern, aber bodenstaendig
- klar auf Anfragen ausgerichtet

## Dateien

- `index.html` Startseite der Demo
- `leistungen.html` Detailseite fuer Leistungen
- `projekte.html` Referenz- und Projektseite
- `kontakt.html` Kontaktseite mit Anfragefokus
- `styles.css` komplettes visuelles System
- `script.js` einfache Reveal-Animationen

## Positionierung

- Branche: Handwerk / Dachdecker
- Demo-Name: Becker Bedachungen
- Fokus: Sanierung, Reparaturen, Neueindeckung
