# Delivery

Hilfswerkzeuge zum Verpacken von Show-Websites als verschickbare Vorschau.

## Beispiel

```powershell
.\ops\delivery\package-preview.ps1 -Source "showcases\handwerk-becker-bedachungen" -Name "handwerk-becker-bedachungen-preview"
```

Das erstellt ein ZIP in `deliveries/`, das du einer Vorschau oder einem Kundenpaket beilegen kannst.
